Install on iPhone

TestFlight
Opens your TestFlight invitation link so testers can install the native iPhone app.

Install on Android

Signed APK
Downloads your signed Android build directly from your website or file host.

How to use it

  • 1
    Upload this page to your website or GitHub Pages.
  • 2
    Set your real TestFlight and APK links in the script below.
  • 3
    Generate one QR code that points to this page.

Good hosting options

GitHub Pages, Netlify, Vercel, or your own domain all work well. For Android, host the APK on the same site, on cloud storage, or on GitHub Releases.

Update the links

Edit the two constants below before publishing. If a link is left empty, its button becomes visually disabled.

const TESTFLIGHT_URL = 'https://testflight.apple.com/join/REPLACE_ME'; const ANDROID_APK_URL = 'https://example.com/gangway-remote.apk';
